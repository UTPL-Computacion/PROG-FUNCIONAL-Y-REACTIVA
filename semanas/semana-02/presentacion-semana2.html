1. Funciones en programación

Definición

Bloques reutilizables que encapsulan lógica. Pueden provenir del lenguaje, bibliotecas o ser creadas por la persona que programa.

Sintaxis (Python)

def saludar(nombre):
    return f"Hola {nombre}"

Uso: saludar("Ana"). La palabra clave return entrega resultados.

Efectos

A diferencia de las funciones matemáticas, pueden producir efectos como imprimir, escribir archivos o cambiar variables externas.

Función vs. método

Función

  • Independiente del estado de un objeto.
  • Se invoca como resultado = funcion(arg).
  • No requiere parámetro self.

Método

  • Asociado a datos/objetos concretos.
  • Se invoca como dato.metodo(arg).
  • Recibe self para acceder al estado.
En PF buscamos funciones desacopladas del estado para facilitar pruebas y reutilización.

2. Funciones puras

Determinismo

Iguales entradas, iguales salidas. Facilita depuración y pruebas.

Sin efectos secundarios

No modifica datos externos ni depende de ellos. Trabaja con parámetros y valores retornados.

Composición

Funciona como bloques LEGO: se pueden encadenar para producir transformaciones complejas.

Relación directa con la inmutabilidad: se evitan datos cambiantes para mantener la pureza.

3. Primeros pasos en PF

Ciudadanas de primera clase

Las funciones se almacenan en variables, se envían como argumentos y se retornan.

Recursión

let rec fib n =
    if n < 2 then 1
    else fib (n-1) + fib (n-2)

Currificación

const sumar = (a) => (b) => a + b;
const sumarTres = sumar(3);

Permite crear funciones especializadas al parcializar argumentos.

4. Ejercicios guiados

Refactorizar suma de pares

  1. Analiza la versión imperativa con bucles y estados mutables.
  2. Identifica patrones de filtrado y acumulación.
  3. Cámbialos por filter y reduce (o foldl en Haskell).

Lambdas con map

Construye una lista de potencias: map(lambda x: 2 ** x, lista).

Experimenta con normalizar cadenas: lambda texto: texto.strip().title().

Documenta las decisiones: ¿cuándo aceptas un efecto secundario? ¿cómo aislas la mutación?
