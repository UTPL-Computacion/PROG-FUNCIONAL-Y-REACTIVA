Información del Curso

Fundamentos de programación funcional y paradigmas

Objetivos

Paradigmas: Imperativo vs Declarativo

Funciones: Puras, de orden superior

Inmutabilidad: Datos inmutables

Lenguajes

Puros: Haskell, Miranda

Impuros: Scala, Python, Java 8+, JavaScript

Históricos: LISP, ML, Erlang

Compendios Semanales

Contenidos organizados cronológicamente por semanas

Semana 01
Paradigmas de Programación
Fundamentos: Paradigmas imperativo vs declarativo, historia de la programación funcional (cálculo lambda, LISP), características clave (inmutabilidad, funciones puras), lenguajes funcionales puros e impuros
Ver compendio Ver presentación
Semana 02
Funciones y Expresiones Lambda
Temario: Funciones desde la matemática hacia la programación funcional, funciones puras y sus características, diferenciación entre funciones y métodos, además de introductorio uso de lambdas y orden superior.
Ver compendio Ver presentación
Semana 03
Recursividad y Listas
Estructuras: Recursión como alternativa a iteración, listas inmutables, operaciones sobre listas (map, filter, reduce), patrones de recursión
Semana 04
Tipos de Datos y Pattern Matching
Tipos: Tipos de datos algebraicos, tipos sum y product, pattern matching, tipos paramétricos y polimorfismo
Semana 05
Evaluación Diagnóstica
Evaluación de conceptos fundamentales de Ingeniería del Software, requerimientos y metodologías ágiles
Semana 06
Diseño de Arquitectura
Unidad 2: DDD ligero, ADRs, monolito modular vs microservicios; contratos y compatibilidad
Semana 07
Modelado UML
Unidad 2: Casos de uso, modelo de contexto, diagramas de clases para el dominio
Semana 08
Diagramas de Interacción y Despliegue
Unidad 2: Secuencia/actividad (escenarios clave), componentes y despliegue orientado a cloud
Semana 09
Git Avanzado y Platform Engineering
Unidad 3: Git avanzado (ramas cortas, PRs, code review), Platform Engineering / IDP y automatización básica
Semana 10
Evaluación Formativa
Evaluación práctica de diseño, arquitectura y herramientas de desarrollo
Semana 11
CI/CD y Seguridad
Unidad 3: CI/CD moderno (pipelines como código, artefactos, feature flags), seguridad: SAST/DAST/SCA, SBOM, firma, supply chain (SLSA)
Semana 12
Contenedores y Observabilidad
Unidad 3: Contenedores, Docker, Kubernetes/GitOps (visión práctica), observabilidad: logs, métricas, trazas (OpenTelemetry), SLO/SLA
Semana 13
Testing y TDD
Unidad 4: Pirámide de pruebas, contratos, mutación y desempeño, TDD/ATDD en el proyecto
Semana 14
IA en Desarrollo de Software
Unidad 4: Asistentes de código, generación de tests y refactor asistido, riesgos (licencias/datos) y verificación humana
Semana 15
Mantenimiento y Legacy
Unidad 4: Mantenimiento, legacy y patrón estrangulador, migraciones, FinOps básico y monitoreo/alertas
Semana 16
Evaluación Sumativa
Evaluación final integral de todos los conceptos y prácticas de Ingeniería del Software
